# Microbiome-targeted nanoplatforms and engineering approaches in breast cancer therapy

![background](./assets/inforgraphics/background.png)

Breast cancer is one of the leading causes of death worldwide, mainly due to its ability to spread (metastasize) and return after treatment. Different types of breast cancer behave differently, making it harder to treat.

Recent studies show that bacteria in the tumor environment (the microbiome) can influence how cancer grows and spreads. Some microbes, like Fusobacterium nucleatum, are linked to worse outcomes.

This work looks at new ways to target these bacteria using nanotechnology, especially tiny particles that can deliver treatment directly to the tumor. These approaches aim to improve diagnosis and make treatments more effective.
